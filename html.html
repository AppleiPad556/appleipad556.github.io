HTML

HTML (Hypertext Markup Language) is one of the fundamental languages in web development.
It operates on opening and closing tags (like <p></p>).
HTML is not case-sensitive, though a general rule of thumb is to write in lowercase.

First Steps:

Begin by including a DOCTYPE call at the top of your HTML document: <!DOCTYPE html>
Then, surround your code with <HTML> tags.
A <head> element determines metadata in the header of a webpage, like the title of the page.
A <body> element is where most of the visible website takes place.
A <footer> element contains the information at the bottom of the website, like copyright information.

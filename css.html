CSS

CSS (Cascading Style Sheets) is a markup language that tells the browser how to style HTML elements.
It can style virtually everything, from colour to fonts to sizes to transitions.
CSS is case-sensitive, so be careful!

Methods:

There are three ways to style HTML elements with CSS.

  1. Inline styles: These styles are directly applied to HTML elements in the HTML document using the style= attribute
  2. Defined styles These styles are defined in the HTML document within <style> elements.
  3. External Spreadsheet: These styles are defined in a different file that is linked to the HTML document.

Syntax:

The syntax for CSS code is:
property: attribute;
For inline styles, this would go right after the style= attribute.
For external spreadsheets, CSS code looks something more like this: 

                        
HTML element {
    property: attribute;
}

for example:

                        
h1 {
    color: purple;
}

There are many elements in CSS that can be used for all sorts of things.
